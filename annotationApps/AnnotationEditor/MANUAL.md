# AnnotationEditor 操作マニュアル

眼筋 MRI アノテーション補正ツール

---

## 画面構成

```
┌─────────────────┬──────────────────────────────────────┐
│   左パネル        │   右パネル（編集エリア）               │
│  ・データ追加      │  ┌─ トップバー（スライス/WL/操作）─┐  │
│  ・AI推論         │  │                               │  │
│  ・ケースリスト    │  │        画像キャンバス           │  │
│  ・保存/完了      │  │                               │  │
│                  │  └───────────────────────────────┘  │
│                  │  ラベルパレット（11ラベル）            │
└─────────────────┴──────────────────────────────────────┘
```

---

## ステータスの種類

| アイコン | 状態 | 説明 | 色 |
|---------|------|------|----|
| ○ | 未推論 (pending) | AI予測なし | グレー |
| ✎ | 編集中 (editing) | 推論済み or 保存済み（完了前） | オレンジ |
| ✓ | 完了 (corrected) | 「完了にする」ボタンで明示的に確定 | 緑 |

- **保存ボタン**を押しても「編集中」のまま
- **完了にするボタン**を押したときのみ「✓完了」になる
- 「✓完了」のケースを再編集して保存すると「✎編集中」に戻る

## データ追加

### 📋 Data.txt からインポート
- `Data.txt` を含む親フォルダを選択
- `series_filter`（config.json）に一致するシリーズを自動検出してNIfTI変換
- ケースID形式: `{PatientID}_{Date}_{Time}_EX1_SE{n}`

### 🏥 DICOM フォルダを直接変換
- IMG番号順のシングルフレーム / Enhanced DICOM フォルダを直接選択して変換

### フォルダ構造
```
app/
  images/                          元画像    {ID}_0000.nii.gz
  output/{model_id}/
    predictions/                   AI予測    {ID}_pred.nii.gz
    edited/                        編集済み  {ID}_corrected.nii.gz
                                             {ID}_edit_log.json
```

---

## AI 推論

### ▶ 一括推論
- `images/` フォルダにある未推論ケースをすべて順番に推論
- GPU（CUDA）が使用可能な場合は自動で GPU を使用、失敗時は CPU にフォールバック
- 推論完了後、ステータスが「✎ 編集中」に更新される

### 推論後に異常排除（チェックボックス）
- デフォルト OFF
- ON にすると推論完了直後に自動で異常排除フィルタを適用

---

## 異常排除フィルタ

推論結果から明らかに誤っているコンポーネントを自動削除します。

### フィルタの種類（適用順）

| フィルタ | 内容 |
|---------|------|
| **小面積除去** | 面積が `min_area_px` 未満のコンポーネントを削除 |
| **端部除去** | 画像端から `edge_margin_pct`% 以内のコンポーネントを削除 |
| **個数上限** | 上記2つ適用後に同ラベルが `max_keep_per_label` 個を超える場合、面積上位のみ残す |

### 設定（config.json で変更可能）
```json
"prediction_filter": {
  "min_area_px": 20,
  "edge_margin_pct": 12,
  "max_keep_per_label": 2,
  "enable_count_filter": true
}
```

### 適用方法
- **手動**: トップバーの **🧹 異常排除** ボタン → 現在ケースの予測ファイルに上書き適用
- **自動**: 左パネルの「推論後に異常排除」チェックボックスを ON にして推論

> ※ フィルタは予測ファイル（`_pred.nii.gz`）を上書きします。元に戻すには再推論が必要です。

## ケースリスト操作

### フィルタボタン
- **すべて** / **未推論** / **✎編集中** / **✓完了** でリストを絞り込み

### 検索
- ケースIDの文字列で絞り込み（部分一致）

### ケース削除
- リストを**右クリック** → 「🗑 リストから削除」（ファイルは残る）

## 画像表示

### ズーム・パン
| 操作 | 動作 |
|------|------|
| マウスホイール | ズームイン / アウト |
| 中ボタンドラッグ | パン（移動） |

### ウィンドウ / レベル（WC / WW）
- デフォルトはケースごとに元画像の最小〜最大値に自動設定
- トップバーのスライダーで調整可能

### マスク表示切り替え
- **Tab キー** または **マスク: ON/OFF ボタン** でトグル

## スライス操作

| 操作 | 動作 |
|------|------|
| ← / → キー | 前 / 次のスライスへ移動 |
| Ctrl + ← | 前スライスのポリゴンをコピー |
| Ctrl + → | 次スライスのポリゴンをコピー |

スライス番号に `*` が付いているスライスは未保存の編集があります。

## アノテーション編集

### ラベル選択

| キー | ラベル | キー | ラベル |
|------|-------|------|-------|
| 1 | ir（下直筋） | 7 | io（下斜筋） |
| 2 | mr（内直筋） | 8 | lac（涙腺） |
| 3 | sr（上直筋） | 9 | on（視神経） |
| 4 | so（上斜筋） | q | ball（眼球） |
| 5 | lr（外直筋） | w | orbit（眼窩） |
| 6 | lev（挙筋）  | 0 | 選択ツール |

選択中のラベルはパレットのボタンがフルカラーで強調されます。

### ポリゴンの描画
1. ラベルを選択（1〜9 / q / w キー）
2. キャンバス上を**左クリック**で頂点を追加
3. **右クリック**でポリゴン確定（3点以上必要）
4. **Esc キー**で描画キャンセル

### ポリゴンの編集（選択ツール: 0）
| 操作 | 動作 |
|------|------|
| 左クリック（ポリゴン上） | 選択（白い頂点が表示） |
| 頂点をドラッグ | 位置を移動 |
| Delete | 選択中のポリゴンを削除 |
| 右クリック | 透明度モードを切り替え |
| Esc | 選択解除 / 描画キャンセル |

### Undo / Redo
- **Ctrl + Z** / **↩ Undo**: 直前の操作を取り消し（最大30ステップ、スライス単位）
- **Ctrl + Y** / **↪ Redo**: やり直し

### リセット
- **リセットボタン**: 現在スライスを AI 予測結果に戻す

## 保存と完了

| ボタン | 動作 | ステータス変化 |
|-------|------|-------------|
| 💾 保存（Ctrl+S） | 全スライスを `_corrected.nii.gz` に保存 | → ✎ 編集中 |
| ✓ 完了にする | 作業完了を明示的に記録 | → ✓ 完了 |

- 「✓完了」のケースを再編集して保存 → 「✎ 編集中」に戻る
- 保存時の通知はポップアップではなく左パネルの小さいテキスト表示

## config.json 設定

```json
{
  "model_id": 119,
  "series_filter": "eT1W_SE_cor",
  "prediction_filter": {
    "min_area_px": 20,
    "edge_margin_pct": 12,
    "max_keep_per_label": 2,
    "enable_count_filter": true
  }
}
```

| 設定 | 説明 |
|-----|------|
| `model_id` | 使用する nnUNet モデルの DatasetID |
| `series_filter` | Data.txt インポート時のシリーズ名フィルタ（部分一致・空なら全シリーズ） |
| `min_area_px` | この面積（ピクセル数）未満のコンポーネントを削除 |
| `edge_margin_pct` | 画像端からこの割合（%）以内のコンポーネントを削除 |
| `max_keep_per_label` | 1ラベルあたりの最大保持個数（面積上位N個を残す） |
| `enable_count_filter` | 個数上限フィルタの有効/無効 |
